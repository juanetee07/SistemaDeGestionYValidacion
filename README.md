## ¿Qué es este proyecto?

<p>
Este sistema permite gestionar <strong>noticias</strong> y <strong>resoluciones académicas</strong> en un entorno digital,
incorporando un modelo de <strong>participación estructurada</strong> donde los usuarios pueden expresar su opinión de forma organizada.
</p>

<p>
No es solo una web informativa — es una herramienta que permite <strong>entender el impacto de las decisiones institucionales</strong>.
</p>

## Objetivo principal

<p>
  Transformar la comunicación institucional en un proceso más claro, participativo y basado en información estructurada.
</p>


## Funcionalidades

<div>

| Módulo                | Qué permite                                      |
| --------------------- | ------------------------------------------------ |
| **Home**              | Información institucional y contexto del sistema |
| **Noticias**          | Publicar y visualizar novedades                  |
| **Resoluciones**      | Gestionar decisiones académicas                  |
| **Comentarios**       | Opiniones estructuradas (no comentarios libres)  |
| **Autenticación**     | Acceso seguro con roles                          |
| **Administración**    | Control y moderación del sistema                 |

</div>
